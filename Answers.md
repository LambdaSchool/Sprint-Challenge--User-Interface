1. If you had to describe semantic HTML to the next cohort of students, what would you say?

a) Semantic HTML wraps content in element tags.  These tags allow a developer to understand how the content should be presented and the tags allow the browser to understand how to display that context to a user.  In addition to assisting a developer, sematic HTML aids in google searches, and allows assistant devices to read web content to a disabled user.

2. Describe some differences between display: block; and display: inline;.

a) Display: block expands the width of the content page. A great example would be a paragraph element. Display: inline, on the other hand, only expands the width of the content like a span element.
